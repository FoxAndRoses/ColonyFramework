About

Colony Framework creates the appearance of an intelligent mining colony through genuine state management and coordination — not simulation shortcuts. Mining stays physical: real drills carving real voxels. The mod discovers ore, plans missions, assigns drones, and flies them through a full mine-and-return cycle, producing colony behavior that emerges from scan data rather than scripted fakery.

It runs as a full Space Engineers mod (not a Programmable Block script) so it can read voxels and coordinate across grids. Every state change is server-authoritative and gated for clean multiplayer behavior, and saves use an append-only schema so old worlds always load safely.

Features

The colony runs itself: survey → mine → produce → print.

▣Colony Identity

A Colony Core block founds a colony with a name, founder, and creation date. Multi-colony support keyed by faction or player.

⛏Autonomous Mining

Full loop: transit → bore with yield-based dynamic depth (drills until the ore runs out) → staged connector docking → cargo unload → draw-derived recharge → automatic re-dispatch. Ice/stone gets ejected on site, not hauled home.

🏭Production Pipeline

Reads the projector blueprint's bill of materials, rolls it up components → ingots → ore against stock, auto-queues assemblers, keeps refineries fed, and orders targeted mining for what's missing.

🔧Ship Printing

Welder drones load components, fly radial outside-in approaches, and weld the blueprint in shipyard stages: frame → internals → hull closure, so interiors go in while the hull is open.

📡Survey Drones

When production needs an ore the colony doesn't know, a scout flies an expanding low-altitude search ring — terrain-following waypoints, one bounded scan per point, coverage persisted across missions.

🛡Obstacle Avoidance

Layered stateless sensing on every transit leg: terrain look-ahead, cheap broad-phase entity queries, and a ray fan only when the corridor is actually blocked. No pathfinding — multiplayer-cheap.

▤Mission System

Mine / Weld / Survey missions with a full claim → assign → in-progress → complete/fail state machine, capability-matched to registered drones.

📊Observability

/colony status live snapshot, HUD GPS markers for every active mission, per-phase log telemetry, and fleet-wide recall.

How it works

The autonomous loop, end to end.

Survey

Detectors + scout drones find deposits; the DB persists.

02

Mine

Drones bore until the ore runs out, dumping stone/ice on site.

03

Deliver

Dock at base, unload cargo, recharge, relaunch — hands-free.

04

Produce

Refineries and assemblers work toward the projected blueprint.

05

Print

Welder drones build the ship: frame → internals → hull closure.
