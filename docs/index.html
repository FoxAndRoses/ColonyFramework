About

Colony Framework creates the appearance of an intelligent mining colony through genuine state management and coordination — not simulation shortcuts. Mining stays physical: real drills carving real voxels. The mod discovers ore, plans missions, assigns drones, and flies them through a full mine-and-return cycle, producing colony behavior that emerges from scan data rather than scripted fakery.

It runs as a full Space Engineers mod (not a Programmable Block script) so it can read voxels and coordinate across grids. Every state change is server-authoritative and gated for clean multiplayer behavior, and saves use an append-only schema so old worlds always load safely.

Features

What's working in-game today.

▣Colony Identity

A Colony Core block founds a colony with a name, founder, and creation date. Multi-colony support keyed by faction or player.

⛏Ore Discovery

Bounded LOD voxel scans, manual (/colony scan) and automatic via round-robin powered ore detectors, with 50 m dedup merge radius.

▤Mission System

One mine mission per unclaimed deposit, with a full claim → assign → in-progress → complete/fail state machine.

⚙Asset Assignment

Register miner grids, detect offline assets, and assign the nearest idle drone to pending missions automatically.

🛰Autonomous Drones

Commission self-test → autopilot transit → gyro-stabilized bore → cargo/timeout watchdogs → retreat climb, all under direct flight control.

📊Resource Tracking

Owner-filtered scan of the colony's grid group totals ore, ingots, and components on a 30-second cadence or on demand.

How it works

The autonomous loop, end to end.

01

Discover

Voxel + ore-detector scans surface rare-ore deposits.

02

Generate

Each unclaimed deposit spawns a mine mission.

03

Assign

Nearest idle drone is matched to the mission.

04

Execute

Transit to standoff, align, and bore to ore depth.

05

Retreat

Climb out, complete the mission, return to idle.
